# SystemVerilog Assertion Learning Sandbox

The primary purpose of this repository,
as a space for code,
is to serve as a sandbox for me to practice learning SystemVerilog
and SystemVerilog Assertions.
Moreover, I can record my thoughts and what I have learned as I go;
giving it a journal quality.

The secondary purpose
(and the reason it is a github repo and not just a folder on my PC),
is as a reference for potential employers to see that these things aren't
entirely alien to me.

## Projects

"Projects" is a bit of a strong word for some of these.
Each of the folders is their own Vivado project,
each with their own README.

In order of creation:
- Blinking LED: This is the equivalent of "hello world"
- Buttons: Bare-bones project to make an interactable program on a FPGA
- Simon Says: A game of Simon Says, my first real project/program
